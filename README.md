# リバーシ
HTMLとCSSで作ったリバーシです。

## 構成ファイル
- `index.html`
- `style.css`
- `script.css`

## 遊び方
「置ける」の部分を長押しすると駒を置けます。

## 注意
- なるべく対策はしましたが不安定で、後半はだいぶ変な挙動になるかもしれません。
- 最新のChromeブラウザを推奨しています。Firefox, Safariでは動かなかったです。(Chrome, Edge, Operaでは動作することを確認できました)
